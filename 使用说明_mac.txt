============================================
  蒲公英达人数据采集工具 — macOS 使用说明
============================================

【适用系统】
  macOS 12+，Intel 和 Apple Silicon (M1/M2/M3) 通用

【运行步骤】

  1. 双击 "启动Chrome.command"
     → 首次运行：右键点击 → 打开 → 确认
     → 会弹出一个新的 Chrome 浏览器窗口
     → 注意：如果旧Chrome已打开，建议先关闭再运行此脚本

  2. 双击 "蒲公英采集"
     → 首次运行：右键点击 → 打开 → 确认
     → 弹出文件选择窗口，选择达人列表 Excel 文件
     → 程序自动检测登录状态
     → 如果没登录，在弹出的 Chrome 窗口里手动登录 蒲公英
       (https://pgy.xiaohongshu.com)
     → 登录后程序自动开始采集
     → 完成后同目录生成 "达人数据.csv"，并自动在 Finder 中定位

【Excel 格式要求】

  第二行为表头，必须有 "Homepage" 列。
  Homepage 列包含小红书主页链接：
    例如：https://www.xiaohongshu.com/user/profile/5a9c1b...

  其他列随意，不影响采集。

【输出说明】

  生成文件：达人数据.csv（可用 Excel / Numbers / WPS 打开）
  列名对齐【Final】2026 爆款种草机 KOL LIST 的表头，可直接粘贴使用。
  自动填充列：
    基础信息：Platform、Influencer name、Followers、Homepage、
             达人所在地、达人类型、合作形式
    报价：平台裸价（图文）、平台裸价（视频）
    数据参考：曝光中位数、阅读中位数、互动中位数、互动率、
             3s阅读率、完播率、活跃粉丝占比、
             观众画像-城市分布Top2、年龄分布Top2、男女比例
  留空列（手动填写）：KPI列、费用列、授权列、推荐理由等

【首次使用注意事项】

  - macOS 安全限制：首次运行时需右键点击文件 → "打开" → 确认
  - 需要在 Chrome 中登录一次蒲公英（登录态会保存在 Chrome 配置中）
  - 采集 50 个达人约需 2 分钟
  - 不要关闭弹出的 Chrome 窗口，采集完成后可关闭

【文件说明】

  启动Chrome.command   双击启动Chrome调试模式
  蒲公英采集            采集程序（双击运行）
  *.xlsx               达人列表（运行时选择）
  达人数据.csv          采集结果（运行后生成）

【常见问题】

  Q: 双击蒲公英采集没反应？
  A: 右键 → 打开 → 确认。首次运行macOS会拦截未签名应用。

  Q: 提示"Chrome未启动"？
  A: 先双击 "启动Chrome.command"，等提示就绪后再运行采集程序。

  Q: Chrome窗口已经打开了但程序说未登录？
  A: 在Chrome中打开 https://pgy.xiaohongshu.com 手动登录即可。

  Q: 数据不对/没数据？
  A: 确认Excel的Homepage列链接格式正确：
     https://www.xiaohongshu.com/user/profile/xxxxx
